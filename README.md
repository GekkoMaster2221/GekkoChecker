## 📖 About

**Gekko Finder** is a security-focused utility designed to help you investigate and verify:

- 🌐 **Websites** — Check if a URL is safe, suspicious, or malicious.
- 📧 **Emails** — Validate email addresses and detect potential phishing sources.
- 🛡️ **Malware & RATs** — Scan for known malware signatures, Remote Access Trojans (RATs), and other threats.

Whether you're a security researcher, developer, or just someone who wants to stay safe online, Gekko Finder gives you the tools to find out what's really going on.
